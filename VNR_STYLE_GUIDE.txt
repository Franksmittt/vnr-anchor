================================================================================
VNR PROFESSIONAL ACCOUNTANTS - DESIGN SYSTEM & STYLE GUIDE
================================================================================

Last Updated: November 2025
Project: VNR Website (Next.js 14 + TypeScript + Tailwind CSS)

================================================================================
1. DESIGN PHILOSOPHY
================================================================================

The VNR website embodies a sophisticated, professional aesthetic that balances:
- Trust & Authority: Deep blues and structured layouts convey expertise
- Approachability: Warm teal accents and clear typography ensure accessibility
- Modern Elegance: Clean lines, subtle animations, and premium spacing
- Strategic Clarity: Content-first design that guides users to key actions

Target Audience: High-net-worth individuals, entrepreneurs, and business owners
across South Africa seeking strategic tax advisory, wealth management, and
business structuring services.

================================================================================
2. COLOR PALETTE
================================================================================

PRIMARY BRAND COLORS
--------------------
Royal Blue (Primary):
  - Base: #234694 - Primary CTAs, icons, accents, main brand color
  - Light: #3d5ba8 - Hover states, secondary actions
  - Dark: #1a3569 - Active states, emphasis, gradients
  - RGB: 35, 70, 148
  - Usage: Used for letters "N", "R", and left stroke of "V" in logo

Lime Green (Secondary):
  - Base: #92C741 - Supporting accents, highlights, checkmark accent
  - Light: #a8d45f - Subtle highlights, decorative elements
  - Dark: #7ab033 - Emphasis, gradients
  - RGB: 146, 199, 65
  - Usage: Used for the checkmark accent that forms the right side of "V" in logo

SURFACE COLORS
--------------
Surface Light: #f8fafc (slate-50)
  - Background for content sections, cards on light backgrounds
  - Provides subtle contrast without harsh whites

Surface Dark: #0f172a (slate-900)
  - Hero sections, footer, navigation
  - Creates depth and premium feel

TEXT COLORS
-----------
Text Primary: #1e293b (slate-800)
  - Main headings and body text on light backgrounds
  - Ensures excellent readability

Text Secondary: #475569 (slate-600)
  - Supporting text, descriptions, metadata
  - Creates visual hierarchy

Text On Dark: #e2e8f0 (slate-200)
  - Text on dark backgrounds (hero, footer)
  - Maintains contrast and readability

GRADIENT USAGE
--------------
CTA Sections: from-brand-blue-dark to-brand-teal-dark
  - Creates dynamic, engaging call-to-action backgrounds using Royal Blue and Lime Green
  - Used sparingly for maximum impact

Hero Overlays: from-slate-900/90 via-slate-900/70 to-slate-800/40
  - Subtle gradient overlays on hero images
  - Ensures text readability while preserving image impact

================================================================================
3. TYPOGRAPHY
================================================================================

FONT FAMILIES
-------------
Sans Serif (Body): Inter
  - Variable: --font-inter
  - Usage: Body text, navigation, UI elements, descriptions
  - Characteristics: Modern, clean, highly readable

Serif (Headings): Lora
  - Variable: --font-lora
  - Weights: 400, 500, 600, 700
  - Usage: All headings (h1-h6), hero text, section titles
  - Characteristics: Elegant, authoritative, adds sophistication

TYPOGRAPHY SCALE
----------------
Hero Headlines (H1):
  - Mobile: text-4xl (2.25rem / 36px)
  - Tablet: text-5xl (3rem / 48px)
  - Desktop: text-6xl (3.75rem / 60px)
  - Weight: font-bold
  - Tracking: tracking-tight
  - Font: Serif (Lora)

Section Headings (H2):
  - Mobile: text-3xl (1.875rem / 30px)
  - Desktop: text-4xl (2.25rem / 36px)
  - Weight: font-bold
  - Font: Serif (Lora)

Subsection Headings (H3):
  - Size: text-2xl (1.5rem / 24px)
  - Weight: font-semibold
  - Font: Serif (Lora)

Card Titles (H3):
  - Size: text-xl (1.25rem / 20px)
  - Weight: font-semibold
  - Font: Serif (Lora)

Body Text:
  - Size: text-base (1rem / 16px) - standard
  - Size: text-lg (1.125rem / 18px) - emphasized
  - Weight: font-normal (400)
  - Font: Sans (Inter)

Small Text / Metadata:
  - Size: text-sm (0.875rem / 14px)
  - Weight: font-medium
  - Font: Sans (Inter)

TEXT HIERARCHY
--------------
- Headings use serif (Lora) for authority and elegance
- Body text uses sans-serif (Inter) for clarity and modernity
- Line height: Default (1.5) for body, tighter for headings
- Letter spacing: tracking-tight for headings, default for body

================================================================================
4. SPACING & LAYOUT
================================================================================

CONTAINER SYSTEM
----------------
Standard Container: container mx-auto px-6
  - Max-width: Responsive (sm: 640px, md: 768px, lg: 1024px, xl: 1280px)
  - Horizontal padding: 1.5rem (24px) on mobile, scales up
  - Centers content with consistent margins

SECTION SPACING
---------------
Standard Section Padding:
  - Mobile: py-20 (5rem / 80px vertical)
  - Desktop: sm:py-28 (7rem / 112px vertical)
  - Creates generous breathing room between sections

Content Spacing:
  - Between elements: mt-6 (1.5rem / 24px) standard
  - Between sections: mt-8 (2rem / 32px) for emphasis
  - Between cards: gap-8 (2rem / 32px) or gap-12 (3rem / 48px)

GRID SYSTEM
-----------
Standard Grid Patterns:
  - 1 column on mobile (grid-cols-1)
  - 2 columns on tablet (md:grid-cols-2)
  - 3 columns on desktop (lg:grid-cols-3)
  - Gap: gap-8 or gap-12 depending on content density

================================================================================
5. COMPONENT PATTERNS
================================================================================

BUTTONS
-------
Primary CTA Button:
  - Background: bg-brand-blue (#234694 - Royal Blue)
  - Text: text-white
  - Padding: px-8 py-3 (horizontal: 2rem, vertical: 0.75rem)
  - Border radius: rounded-md or rounded-full
  - Hover: hover:bg-brand-blue-light, hover:scale-105
  - Shadow: shadow-lg
  - Transition: transition-transform duration-300

Secondary Button:
  - Background: bg-white or transparent
  - Border: border border-slate-300
  - Text: text-text-primary
  - Hover: hover:border-brand-blue, hover:text-brand-blue

Rounded Full Button (Special CTAs):
  - Border radius: rounded-full
  - Used for prominent call-to-actions
  - Often includes icons (ChevronRight, ArrowUpRight)

CARDS
-----
Standard Card:
  - Background: bg-white
  - Border: border border-slate-200
  - Border radius: rounded-2xl (1rem / 16px)
  - Shadow: shadow-sm (subtle) to shadow-lg (prominent)
  - Padding: p-6 to p-8
  - Hover: hover:shadow-2xl, hover:-translate-y-2 (lift effect)

Dark Card (Service Cards):
  - Background: bg-surface-dark (#0f172a)
  - Text: text-white
  - Border: border-slate-700
  - Creates premium, sophisticated feel

Icon Cards:
  - Icon container: rounded-full, bg-brand-blue
  - Icon size: h-8 w-8 to h-10 w-10
  - Positioned absolutely above card (translate-y-1/2)

HERO SECTIONS
-------------
Structure:
  - Full viewport height: min-h-[calc(100vh-theme(spacing.20))]
  - Background image with gradient overlay
  - Content: max-w-3xl container
  - Text: White on dark gradient
  - CTA: Prominent primary button

Gradient Overlay:
  - from-slate-900/90 via-slate-900/70 to-slate-800/40
  - Ensures text readability over images

NAVIGATION
----------
Header:
  - Background: bg-surface-dark
  - Height: h-20 (5rem / 80px)
  - Sticky: sticky top-0 z-50
  - Shadow: shadow-lg

Navigation Items:
  - Active: bg-brand-blue-dark text-white
  - Inactive: text-text-on-dark/80 hover:bg-white/10
  - Border radius: rounded-md
  - Padding: px-4 py-2

Mobile Menu:
  - Backdrop: bg-black/50 backdrop-blur-sm
  - Panel: bg-surface-light, max-w-xs
  - Slide-in from right

================================================================================
6. ANIMATIONS & INTERACTIONS
================================================================================

SCROLL ANIMATIONS
-----------------
Fade In Up:
  - Keyframe: fadeInUp
  - Duration: 0.8s
  - Easing: ease-out
  - Effect: Fades in while moving up 20px

Reveal:
  - Keyframe: reveal
  - Duration: 1s
  - Easing: cubic-bezier(0.215, 0.610, 0.355, 1)
  - Effect: Fades in, moves up 40px, scales from 0.98, blur to clear

Staggered Delays:
  - First element: 0ms
  - Second: 200ms
  - Third: 400ms
  - Creates cascading reveal effect

HOVER EFFECTS
-------------
Card Lift:
  - Transform: hover:-translate-y-2 (lifts 8px)
  - Shadow: hover:shadow-2xl (increases shadow)
  - Border: hover:border-brand-blue/30 (adds color)
  - Duration: transition-all duration-300

Button Scale:
  - Transform: hover:scale-105 (5% larger)
  - Duration: transition-transform duration-300

Icon Color Change:
  - Icons change color on hover (e.g., slate-400 to brand-blue)

7. TONE OF VOICE
================================================================================

OVERALL VOICE
-------------
Professional yet Approachable
  - Authoritative without being condescending
  - Expert knowledge communicated clearly
  - Respectful of client intelligence

Strategic & Forward-Looking
  - Focus on long-term value, not just compliance
  - Proactive rather than reactive
  - Solution-oriented language

Trustworthy & Transparent
  - Clear, honest communication
  - No jargon without explanation
  - Direct about capabilities and value

WRITING STYLE
-------------
Headlines:
  - Action-oriented and benefit-focused
  - Examples: "Anchor-backed investing. VNR-led tax strategy."
  - "Optimize Your Tax Burden. Maximize Your Returns."
  - "Build a Legacy. Protect Your Future."

Body Copy:
  - Clear, concise sentences
  - Avoid unnecessary jargon
  - Use active voice
  - Break complex concepts into digestible points

Service Descriptions:
  - Lead with client benefit
  - Follow with technical capability
  - End with reassurance or next step

Call-to-Actions:
  - Direct and action-oriented
  - Examples: "Schedule a Strategic Consultation"
  - "Explore Anchor Wealth Division"
  - "Book a Will & Risk Review"

KEY MESSAGING THEMES
--------------------
1. Strategic Partnership (not just service provider)
2. Long-term Legacy Building (beyond immediate needs)
3. Nationwide Service (while head office in Centurion)
4. Tax Efficiency & Wealth Protection
5. Expert Guidance with Personal Touch

================================================================================
15. DO'S AND DON'TS
================================================================================

DO:
- Use serif fonts for all headings
- Maintain generous spacing between sections
- Use brand blue for primary actions
- Include hover states on all interactive elements
- Ensure text is readable on all backgrounds
- Use rounded-2xl for cards (premium feel)
- Include icons to break up text
- Use gradient backgrounds sparingly (CTAs only)

DON'T:
- Mix font families incorrectly (serif for body, sans for headings)
- Use too many colors (stick to brand palette)
- Overcrowd sections with content
- Use flat buttons without hover states
- Place text over images without gradient overlay
- Use sharp corners on cards (use rounded-2xl)
- Overuse animations (subtle is better)
- Compromise readability for style
